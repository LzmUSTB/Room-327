# Room-327

功能：创建/加入局域网聊天室，可以在聊天室的大厅中发送群体信息，或跟聊天室的任意其他用户私聊。

•应用框架：**Electron** v28.2.0

•前端框架：**Vue** v3.4.15

•项目脚手架：**Vite** v5.0.12 

•通信协议：**Websocket** v8.16.0

## 项目初始化

### 安装

```bash
$ npm install
```

### 运行

```bash
$ npm run dev
```

### 构建

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
